Threat Categories affecting species

As a part of the Hack The Redlist hackathon, we worked together to create a visualisation of the threats that species face based on their endangerment level.

These visualizations tell an interesting tale about the threats that face some of the most threatened species on our planet, as well as those for whom time has already run out. The results back up commonly held beliefs as to what makes species vulnerable to extinction.

vulnerable

Our visualization suggests that threats to species in these categories are linked to habitat changes. One of the prime examples is forest area loss due to deforestation for logging, plantations or mining. However it also looks like animal trade, poaching and hunting are key factors.

Project Summary

The data that we used is located in the public repo's data folder. Tom used the data from threat.json to create a scaled list of the most frequent words in thread_txt by category. The R script for the filtering is available in this same folder. Then based on the exported csvs, Sandra turned those back to a JS object and used a publicly available wordcloud project to display them on this website.

After the hackathon, we added some explanatory text to the results and a bit more context to the project as well as styling that makes the project a bit more friendlier to anyone who visits the page. Sandra also included some sharing options.

Some of the enhancements we thought this project would benefit from:

  • It would be great to color the words in the same as the buttons at the top (or shades of that colour)
  • It would be great to have a cool transition when moving from one category to another, perhaps similar to the animation you get when re-sizing the window
  • Shape the word cloud according to the letters of the classification or in the silhouette of an animal in that category.

If you'd like to help out, send a pull request for these changes, or fork the project to come up with something even more useful and fun!
